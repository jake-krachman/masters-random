# Masters Vault Randomizer

[https://masters-vault-random.com](https://masters-vault-random.com/)

A simple site that shuffles through nearly 10,000 clips from the Masters Tournament Video Vault (1968-2025). Click shuffle to get a random moment from Masters history.

## Features

- Random clip selection from 9,895 vault clips spanning 58 years
- Draggable year range slider to filter by era
- Direct links to watch clips on masters.com
- Global shuffle counter across all visitors

## Notes

- You must be logged into [masters.com](https://www.masters.com) to watch clips
- This project is not affiliated with or endorsed by Augusta National Golf Club or the Masters Tournament
